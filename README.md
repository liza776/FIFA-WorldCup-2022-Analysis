# ⚽ FIFA World Cup 2022 Analysis

### Project Overview
Exploratory Data Analysis (EDA) of the 2022 World Cup using SQL to rank teams by goals, assists, and discipline.

### Key Features
* **Team Stats:** Analyzed possession, passing accuracy, and age distribution.
* **Offensive/Defensive Metrics:** Ranked teams by goals per 90 minutes and expected goal differences.
* **Group Stage Analysis:** Summarized performance metrics for each tournament group.

### Tech Stack
* **Database:** MySQL
* **Key Skills:** Aggregations, Multi-table Joins, and Performance Ranking.
